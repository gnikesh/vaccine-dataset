# Vaccine Tweets Dataset

This repository contains dataset and code used for our paper *"The Shifting Landscape of Vaccine Discourse: A Massive Dataset Examining a Decade of Social Media Posts on Vaccines Before and During the COVID-19 Pandemic"*.

The ```dataset/vaccines_tweet_ids``` folder contains ```.zip``` files with tweet ids of vaccine tweets from years 2013-2022. The ```dataset/control_medical_tweet_ids``` contains the tweet ids of tweets with medical terms. 

The ```code``` folder contains the code used for computing the UED metrics. The code is taken from [Emotion Dynamics](https://github.com/Priya22/EmotionDynamics/tree/master). If you use any of the resources in the code directory, cite the following:
